
# Getting and Cleaning Data Project

## Files
- run_analysis.R: script that processes raw data into tidy data
- tidy_data.txt: final output dataset

## How to run
1. Download dataset and unzip
2. Set working directory to "UCI HAR Dataset"
3. Run run_analysis.R
4. Output file tidy_data.txt will be created

## Output
A tidy dataset with the average of each variable for each activity and subject.
